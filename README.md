# Multilingual Voice-to-Voice Translator

This project captures speech, translates it to multiple languages at the same time and plays the translated audio back. It combines speech-to-text, translation, and text-to-speech to create a real-time multilingual communication tool.

## Features
- Speech input via microphone
- English speech transcription (AssemblyAI)
- Translation to Arabic, Chinese, and Urdu
- Voice output using ElevenLabs
- Gradio interface for easy interaction

## Tech Stack
- Python
- AssemblyAI (Speech-to-Text)
- Translate library (Text Translation)
- ElevenLabs API (Text-to-Speech)
- Gradio UI
